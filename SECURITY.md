# Security Policy

## Supported Versions
OSE is currently maintained on the `main` branch only.

## Reporting a Vulnerability
Please avoid opening public issues for security-sensitive problems.
Instead, contact the repository owner privately and include:
- affected area
- reproduction steps
- impact assessment
- suggested mitigation if available

## Operational Notes
- Default credentials are intended for local demonstration only.
- Change `JWT_SECRET` and default admin credentials before exposing the stack outside a trusted environment.
- Do not commit real production secrets into this repository.
