# OSE 初始化说明

## 1. 初始化目标
首次启动系统时，自动完成以下动作：
- 创建默认管理员账号
- 初始化系统设置（考试日期、通过分数、学习偏好、复习周期）
- 写入软件设计师备考知识树
- 写入自拟题库示例数据
- 自动生成当前学习计划
- 初始化练习记录、错题记录、模拟考试和示例笔记

## 2. 触发时机
- 后端服务启动后，由 `backend/src/main/java/com/ose/bootstrap/DataInitializer.java:1` 自动执行。
- 数据初始化基于“若为空则插入”的幂等策略，不会在每次重启时覆盖已存在的数据。

## 3. 默认数据内容
### 管理员账号
- 用户名：`admin`
- 密码：`OseAdmin@2026`
- 角色：`ADMIN`

### 系统设置默认值
- 考试日期：`2026-05-23`
- 通过分数：`45`
- 每周学习时长：`12`
- 学习偏好：工作日专题学习，周末整卷模拟
- 默认复习周期：`1,3,7,14`
- 单次学习时长：`90` 分钟

### 示例业务数据
- 多级知识点树
- 上午单选题示例
- 下午案例题示例
- 当前学习计划与任务
- 示例练习会话与错题
- 上午 / 下午模拟卷及成绩记录
- Markdown 学习笔记

## 4. 初始化验证
启动完成后，可通过以下方式验证：
```bash
curl http://localhost/healthz
curl http://localhost:8080/api/health
```
然后使用默认管理员账号登录前端。

## 5. 重置初始化数据
如需回到全新状态：
```bash
docker compose down -v
docker compose up -d --build
```
该操作会清空 MySQL 卷并重新触发初始化。
